# SmartHelmet Mobile App

A Flutter + Kotlin application for the SmartHelmet ESP32 safety helmet system with persistent Bluetooth Classic connection and background monitoring.

Features

1.Real-time Sensor Dashboard** - Live acceleration & gyroscope readings  
2.Accident Detection & Alerts** - Immediate notifications with severity levels  
3.**----Background Bluetooth Service** - Persistent connection even when app minimized  
4.**---Emergency SMS Integration** - Auto-send SMS to emergency contacts  
5.**---GPS Location Tracking** - Record accident location  
6.**--Accident History Log** - Offline database storage  
7.**---Helmet Status Monitor** - Real-time helmet-worn detection  
  **---Android Optimized** - Foreground service for reliability  

##Prerequisites

--Flutter SDK**: 3.0+
--Android SDK**: API 21+ minimum
- **Kotlin**: 1.7+
- **Android Studio**: Latest version
- **Physical device or emulator**: Android 5.1+
